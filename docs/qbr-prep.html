Share With Your Team
Mac / Linux

Download install.sh from SharePoint Team Internal/ and run:

bash install.sh
What it does:  ✅ Installs /qbr-prep  ·  ✅ Installs /deck-builder  ·  ✅ Installs the PPTX engine  ·  ✅ Installs python-pptx  ·  ✅ Guides through Microsoft 365 MCP setup
Windows — PowerShell

Run once in PowerShell from the project folder:

PS> New-Item -ItemType Directory -Force "$env:USERPROFILE\.claude\commands"
PS> Copy-Item ".claude\commands\qbr-prep.md" "$env:USERPROFILE\.claude\commands\qbr-prep.md"
PS> Copy-Item ".claude\commands\deck-builder.md" "$env:USERPROFILE\.claude\commands\deck-builder.md"
No terminal? Use File Explorer — paste the .md files into C:\Users\<you>\.claude\commands\ (create the folder if it doesn't exist).
